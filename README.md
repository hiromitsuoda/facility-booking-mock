# 施設予約管理システム — フロントエンドモック

ひかり市「ひかりプラザ・セントラル」を想定した、施設予約管理システムのフロントエンドモックアプリです。

## 技術スタック

- [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) — アイコン

## 画面一覧

| 画面 | 説明 |
|------|------|
| 施設一覧・検索 | 絞り込みパネル（目的・場所 / 利用日時 / 時間帯）＋施設カード＋室場リスト |
| 施設の空き状況 | 月カレンダー形式で○×等のステータス表示、凡例パネル付き |
| 予約登録（モーダル） | 日付クリックで開く予約フォーム（バリデーション・送信完了画面あり） |

## ローカル起動

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
npm run build
```
